Getting Better Answers

Here are a few small things that we built, that you also might enjoy:

  • Full Markdown Support (Add "Format as markdown" to your prompt)
  • Neat Tables ("Render as tabular data")
  • Mermaid Graphs ("Use mermaid.js to describe the following...")
  • Code Syntax Highlighting
  • Export Everything to Markdown (Go to "Settings > Updates > Save History"
  • Custom CSS (Go to "Settings > Theme")

We'll document more features as we move forward, so stay tuned and make sure you've got "Automatically Check for Updates" enabled in your Settings!
